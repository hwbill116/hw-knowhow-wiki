# 🧠 个人知识体系

本知识库由 AI Agent 自动维护，使用 obsidian-wiki 框架构建。

## 核心模块

- [[采集]] — 从网页、文档、GitHub 等来源获取知识
- [[整理]] — 结构化存储、分类、关联知识
- [[输出]] — 生产文档、报告、代码等
- [[进化]] — 持续学习、纠错、改进

## 工具链

- **obsidian-wiki** — AI Agent 自动生成双链笔记
- **obsidian-digital-garden** — 发布为网页
- **Agent Memory** — 跨会话持久记忆
- **Ontology** — 类型化知识图谱
